# Zu entfernende Phrasen

## Ankündigungs-Einleitungen

Diese Ankündigungsphrasen entfernen. Den Inhalt direkt nennen.

- "Folgendes ist wichtig:"
- "Folgendes gilt es zu beachten:"
- "Hier ist, was [X]"
- "Hier ist, warum [X]"
- "Das Unbehagliche daran ist"
- "Es stellt sich heraus"
- "Die eigentliche [X] ist"
- "Um das klarzustellen"
- "Die Wahrheit ist,"
- "Ich sage es noch einmal:"
- "Um ehrlich zu sein"
- "Können wir über [X] reden"
- "Was ich interessant finde:"
- "Das eigentliche Problem ist"
- "Ich muss das ansprechen:"

Jede "Hier ist was/das/warum"-Konstruktion ist eine Einleitung vor dem eigentlichen Punkt. Streichen und direkt zum Punkt kommen.

## Betonungskrücken

Diese fügen keine Bedeutung hinzu. Einfach löschen.

- "Punkt." / "Ende."
- "Lass das sacken."
- "Das ist wichtig, weil"
- "Kein Zweifel"
- "Deshalb ist das relevant"

## Geschäftsjargon

Durch klare Sprache ersetzen.

| Vermeiden | Stattdessen |
|-----------|-------------|
| Navigieren (Herausforderungen) | Umgehen, angehen |
| Aufschlüsseln (Analyse) | Erklären, untersuchen |
| Einlassen auf | Annehmen, umarmen |
| Landschaft (Kontext) | Situation, Bereich |
| Gamechanger | Bedeutend, wichtig |
| Verdoppeln | Festhalten, erhöhen |
| Tieftauchen | Analyse, Untersuchung |
| Einen Schritt zurücktreten | Überdenken |
| In Zukunft | Als Nächstes, ab jetzt |
| Zurückkommen auf | Zurückkehren zu, erneut betrachten |
| Auf einer Linie sein | Einig, abgestimmt |
| Synergien nutzen | Zusammenarbeiten |
| Transparenz zeigen | Offen sein |
| Mehrwert schaffen | Nützlich sein |

## Adverbien

Alle Adverbien eliminieren. Keine Abschwächer, keine Verstärker, keine Hecken.

Konkrete Täter:

- "wirklich"
- "einfach"
- "buchstäblich"
- "aufrichtig"
- "ehrlich gesagt"
- "schlicht"
- "eigentlich"
- "tief"
- "wahrhaftig"
- "grundlegend"
- "inhärent"
- "zwangsläufig"
- "interessanterweise"
- "wichtigerweise"
- "entscheidenderweise"

Auch diese Füllphrasen streichen:

- "Im Kern"
- "In der heutigen [X]"
- "Es ist erwähnenswert"
- "Am Ende des Tages"
- "Wenn es um [X] geht"
- "In einer Welt, in der"
- "Die Realität ist"

## Meta-Kommentar

Selbstreferenzielle Einschübe entfernen. Der Text soll sich bewegen, nicht seine eigene Struktur ankündigen.

- "Tipp:"
- "Überraschung:" / "Spoiler:"
- "Das weißt du bereits, aber"
- "Das ist ein anderer Artikel"
- "X ist ein Feature, kein Bug"
- "Verkleidet als"
- "Der Rest dieses Textes erklärt..."
- "Lass mich dich durchführen..."
- "In diesem Abschnitt werden wir..."
- "Wie wir sehen werden..."
- "Ich möchte erforschen..."

## Performative Betonung

Falsche Intimität oder konstruierte Aufrichtigkeit:

- "schleicht sich ein"
- "Ich verspreche es"
- "Sie existieren, versprochen"

## Ankündigen statt Zeigen

Schwierigkeit oder Bedeutung ankündigen statt zeigen:

- "Das ist wirklich schwer"
- "So sieht echte Führung aus"
- "So sieht [X] in der Praxis aus"
- "wirklich relevant ist"

## Vage Aussagen

Sätze, die Wichtigkeit ankündigen, ohne das Spezifische zu benennen. Streichen.

- "Die Gründe liegen in der Struktur"
- "Die Implikationen sind erheblich"
- "Das ist das tiefste Problem"
- "Die Einsätze sind hoch"
- "Die Konsequenzen sind real"

Wenn ein Satz etwas als wichtig/tief/strukturell bezeichnet, ohne das Spezifische zu zeigen: streichen oder durch das Spezifische ersetzen.
